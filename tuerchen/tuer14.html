Türchen 14: Schleifen in R

Heyy, da wir gerade erst die R-Objekte wiederholt haben, können wir uns noch kurz die Schleifen anschauen. Wir starten mal mit der gängigsten, die du wahrscheinlich auch in 99% der Fälle benutzen wirst. Die "for-Schleife". Die Syntax ist ganz easy:

for (i in 1:10){
    print(i)
}

Wir starten immer mit einen for und dahinter unsere Laufvariable die sich verändert i kombiniert mit den Wertebereich welchen i annehmen soll. Den simplesten Fall siehst du hier oben schon, bei dem wir einen Vektor 1:10 als Bereich auswählen. Oft ist es aber so das wir die Grenzen nicht wirklich kennen und das ganze dynamischer machen müssen. Wir fangen dabei simpel an:

zahlen = c(10, 20, 30, 40)

Deine Task wäre es, den Index und passend dazu den Wert auszugeben, wobei der code auch funktionieren soll wenn sich der Vektor in der größe verändert.

Schleifen - Seite 2

Lösung:

for (i in 1:length(zahlen)) {
    cat("Index:", i, "| Wert:", zahlen[i],"\n")
}

Wennnn wir jetzt mit Matrizen oder Dataframes arbeiten müssen, wäre es auch gut das dynamisch zu machen. Da wir aber erstmal nur eine for-Schleife haben, wäre die Task für:

mat = matrix(1:9, nrow = 3, byrow = TRUE)

daten = data.frame(
    ID = c("Seq1", "Seq2", "Seq3"),
    Länge = c(1500, 2300, 1890),
    GC_Gehalt = c(0.45, 0.52, 0.47)
)

Printe die Zeilen der Matrix und die des Dataframes "schön" aus, jeweils mit einer for-Schleifeee

Schleifen - Seite 3

Lösung:

for (i in 1:nrow(mat)) {
    cat("Zeile", i, ":", mat[i, ],"\n")
}

for (i in 1:nrow(daten)) {
    cat("ID:", daten$ID[i], "| Länge:", daten$Länge[i],"| GC-Gehalt:", daten$GC_Gehalt[i], "\n")
  }

Als kleines Addon, man muss auch nicht immer mit indexen arbeiten, wir können auch einfach direkt über einen Vektor laufen:

sammlung = c("Apfel", "Banane", "Kirsche")

Versuch mal den Vektor selbst als Wertebereich zu nehmen und dann die Elemente auszugeben.

Schleifen - Seite 4

Lösung:

for (frucht in sammlung) {
    cat("Frucht:", frucht,"\n")
}

Okeee, super das wären jetzt so die Basic sachen die man kennen sollte, als nächstes wollen wir uns aber noch Verschachtelte for-Schleifen anschauen. Deine Aufgabe wäre es mithilfe von 2 for-Schleifen die Position z.B. [1,1] und passend dazu den Wert für jeden Eintrag in der Matrix auszugeben, die wir vorhin erstellt haben. Wichtig dabei ist das die Laufindexe in den beiden Schleifen unterschiedlich sein müssen. Warum auch immer fängt man immer mit i an und geht alphabetisch weiter also j. Du musst auch noch beachten das die geschweifte Klammer der ersten Schleife, die zweite umschließt, und entsprechend die zweite Schleife, einmal komplett durchläuft bis die erste Schleife wieder dran ist mit den nächsten Wert. Aber jaaa, das kriegst du hin :]

Schleifen - Seite 5

Lösung:

for (i in 1:nrow(mat)) {
    for (j in 1:ncol(mat)) {
        cat("Element bei [", i, ",", j, "] ist", mat[i, j],"\n")
    }
}

Alsss nächstes können wir noch so stuff anschauen den man häufig in Verbindung mit schleifen macht, und das wäre das arbeiten mit "Conditions". Also das wir bestimmte Dinge nur tun, wenn der Laufindex es erfüllt. Du kannst mal ganz Basic, für die Zahlen von 1 bis 10 ausgeben, ob sie gerade/ungerade sind, als tipp %% ist praktisch.

Schleifen - Seite 6

Lösung:

for (i in 1:10) {
    if (i %% 2 == 0) {
        cat(i, "ist gerade\n")
    } else {
        cat(i, "ist ungerade\n")
    }
}

Zuletzt können wir das noch mit break und next Befehlen koppeln. Versuch diesmal nur die geraden Zahlen von 1 bis 10 auszugeben, mithilfe von next. Unnnnd versuch mal, obwohl wir eigentlich bis 10 laufen wollen, schon bei 8 abzubrechen, mit dem break Aufruf.

Schleifen - Seite 7

Lösung:

for (i in 1:10) {
    if (i == 8) {
        cat("Wir stoppen schon bei 8")
        break
    }
    if (i %% 2 != 0) {
        next
    }
    cat(i, "ist gerade \n")
}

Supperrr, wir sollten jetzt das wichtigste zu for-Schleifen abgedeckt haben, morgen schauen wir uns noch die repeat und while schleife an. Die nutzt man nicht wirklich, aber falls Sie in der Klausur drankommen in vorgeschriebenen Code, wäre es trotzdem gut das Konzept dahinter zu verstehen, alsoooooooo hoffentlich bist du jetzt sicherer in for-Schleifen und tschauuu :]
