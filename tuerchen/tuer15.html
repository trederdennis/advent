Türchen 15: Weitere Schleifen in R

Ellooo, heute schauen wir uns die while und repeat Schleifen an. Wir benutzen sie, wenn wir nicht wissen, wie viele Iterationen wir benötigen. Wir starten mal mit der while Schleife. Die grundlegende Syntax sieht so aus:

while (Bedingung) {
    # Code, der ausgeführt wird, solange die Bedingung wahr ist
}

Ein einfaches Beispiel wäre:

zahl = 1
while (zahl <= 5) {
    print(zahl)
    zahl = zahl + 1
}

Wir definieren unsere Laufvariable hier immer außerhalb, also hier zahl. Solange die Bedingung zahl <= 5 wahr ist, wird der Code in der Schleife ausgeführt. Bei jeder Iteration erhöhen wir zahl und sobald es größer als 5 ist, stoppt die Schleife. Die Condition sind dabei die typischen logischen mit ">,<,==,!=".

Deine Task wäre es eine while-Schleife zu erstellen, die so lange Zahlen ausgibt, bis sie größer als 100 sind. Starte bei 1 und verdopple die Zahl in jeder Iteration.

While-Schleifen - Seite 2

Lösung:

zahl = 1
while (zahl <= 100) {
    cat("Wert:", zahl, "\n")
    zahl = zahl * 2
}

Wenn du jetzt bspw. mit Vektoren arbeitest könntest du die typischen length() Bedingungen usen und hättest jetzt ein bisschen mehr Spielraum wie du das Indexen organisieren könntest. Alsoooo, du kannst hier dadurch das der Laufindex außerhalb definiert ist, ihn dynamisch wieder kleiner machen oder in variablen größen erhöhen, aber oft will man auch einfach nur konstant laufen und dann ist es auch nur ein gimmick. Das wichtigste ist, dass die Bedingung irgendwann FALSE wird. Sonst haben wir eine Endlosschleife.

Du kannst mal versuchen wie bei den for-Schleifen eine while-Schleife zu bauen die alle Geraden Zahlen von 1-20 ausgibt bzw. bei 15 einfach aufhört, dafür brauchst du wieder:

  • alle ungeraden Zahlen überspringen mit next
  • Wenn wir bei 15 sind, mit break aufhören

While-Schleifen - Seite 3

Lösung:

zahl = 1
while (zahl <= 20) {
    if (zahl == 15) {
        cat("Wir stoppen bei 15\n")
        break
    }
    if (zahl %% 2 != 0) {
        zahl = zahl + 1
        next
    }
    cat("Gerade Zahl:", zahl, "\n")
    zahl = zahl + 1
}

So als kleine Challenge, kannst du mal überlegen wie du "Verschachtelte for-Schleifen" mit nur einer while-Schleife umsetzen könntest. Also diesen Code:

mat = matrix(1:9, nrow = 3, byrow = TRUE)

for (i in 1:nrow(mat)) {
    for (j in 1:ncol(mat)) {
        cat("Element bei [", i, ",", j, "] ist", mat[i, j],"\n")
    }
}

Ist wirklich nicht so einfach, wenn dir nicht direkt eine Idee einfällt, kannst du dich in die Lösung reindenken, ist nur so ein Beispiel, an den man zeigen kann wie flexibel man sein kann, beim Laufindex.

While-Schleifen - Seite 4

Lösung:

index = 1
n = length(mat) 

while (index <= n) {
    i = ceiling(index / ncol(mat))
    j = index %% ncol(mat) 
    
    if (j == 0){
      j = ncol(mat)
    }
    
    cat("Element bei [", i, ",", j, "] ist", mat[i, j], "\n")
    
    index = index + 1
}

Das war denk ich mal alles wichtige zu while-Schleifen. Man used halt öfter for-Schleifen, weil man da nicht die Gefahr hat in Endlosschleifen zu fallen, aber weiter gehts auf der nächsten page mit repeat.

Repeat-Schleifen - Seite 5

Die repeat-Schleife läuft im Vergleich zur while-Schleife immer, bis sie explizit mit break gestoppt wird.

Die grundlegende Syntax sieht so aus:

repeat {
    # Code, der ausgeführt wird
    if (Bedingung) {
        break
    }
}

Wieder ein einfaches Beispiel:

zahl = 1
repeat {
    cat("Wert:", zahl, "\n")
    zahl = zahl * 2
    if (zahl > 100) {
        break
    }
}

Diesen typ von Schleife nutzt man halt wirklich gar nicht, alsooooo hab ich mir gedacht das du vllt. als Übung versuchts für diesen Code:

repeat {
    dna = paste0(sample(c("A", "T", "C", "G"), 10, replace = TRUE), collapse = "")
    cat("Generierte DNA-Sequenz:", dna, "\n")
    if ( ? ) {
        cat("Sequenz enthält 3 As. Schleife wird beendet.\n")
        break
    }
}

Eine Abbruchbedingung zu definieren, die stoppt sobald "AAA" in der generierten Sequenz zu finden ist. Als Tipp brauchst du grepl().

Repeat-Schleifen - Seite 6

Lösung:

repeat {
    dna = paste0(sample(c("A", "T", "C", "G"), 10, replace = TRUE), collapse = "")
    cat("Generierte DNA-Sequenz:", dna, "\n")
    if (grepl("AAA", dna)) {
        cat("Sequenz enthält 3 As. Schleife wird beendet.\n")
        break
    }
}
